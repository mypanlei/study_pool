# 飞书多维表格系统介绍

> 飞书多维表格 (Base/Bitable) 是介于电子表格、数据库、项目管理工具和低代码应用平台之间的业务协作工具，提供字段类型、多视图、表间关联、权限、自动化、仪表盘和开放 API。

## 核心贡献

1. **"多维"的含义**：字段类型多维（文本/数字/日期/人员/关联/公式/AI 等）、数据关系多维（表间关联）、展示视图多维（表格/看板/甘特/日历/表单）、协作权限多维、流程多维（自动化触发）。
2. **基础数据模型**：多维表格 App → 数据表 → 记录/字段 → 视图 → 仪表盘 → 自动化流程。
3. **核心能力模块**：字段系统（20+ 字段类型）、视图系统（6 种视图）、表间关联、表单收集、自动化流程（触发+操作）、权限系统（源表+应用双层）、仪表盘分析、开放 API。
4. **典型场景**：项目管理、CRM 客户管理、工单与需求管理、内容生产管理、招聘流程管理、库存与资产管理。
5. **与普通电子表格的本质区别**：电子表格偏计算工具，多维表格偏业务系统（结构化、协作、权限、自动化、集成）。
6. **建模方法**：识别业务对象 → 拆分数据表 → 定义字段类型 → 建立表间关联 → 设计视图 → 设计权限 → 添加自动化 → 接入 API。
