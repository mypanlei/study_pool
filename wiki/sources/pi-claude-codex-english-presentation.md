# Pi vs Claude Code vs Codex — English Presentation

> An English corporate presentation comparing three coding agents: Pi (open multi-provider harness), Claude Code (integrated product), and Codex (cloud-based engineering agent).

## Core Arguments

1. **Pi** — Open-source coding agent harness, multi-provider, local CLI. Best when openness and provider flexibility matter most.
2. **Claude Code** — Integrated agentic coding product for day-to-day development across terminal, IDE, desktop, and web.
3. **Codex** — Cloud-based software engineering agent for delegated, parallel, and auditable execution in isolated sandboxes.
4. **Key distinction** — The biggest difference is not model quality, but where work runs, who controls the environment, and how tasks are executed.
5. **Selection rule** — Control first → Pi; Daily flow first → Claude Code; Delegation first → Codex.
