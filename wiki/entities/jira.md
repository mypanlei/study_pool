# Atlassian Jira

> Atlassian 出品的专业软件研发项目管理和 Issue 跟踪系统，以 Issue 工单模型为核心，提供 Sprint/Backlog/工作流/版本管理/研发报表等能力。

## 关键信息

- **类型**: 项目管理 / Issue 跟踪 / 研发流程管理
- **开发商**: Atlassian
- **核心模型**: Issue（需求/Story/Task/Bug/Epic/Sub-task）
- **配套工具**: Confluence（文档）、Bitbucket（代码）、Opsgenie（告警）
- **典型场景**: 软件研发管理、缺陷跟踪、敏捷迭代、IT 服务管理、DevOps 协同

## 核心特色

1. **Issue 工作流引擎**：严格状态机控制（待处理→处理中→审查→测试→完成），可配置状态、流转条件、权限和必填字段。
2. **敏捷研发管理**：Scrum Board/Kanban Board/Backlog/Sprint/Epic/Version 完整研发管理链路。
3. **研发报表**：燃尽图、速度图、控制图、累计流图、版本进度、缺陷趋势。
4. **集成生态**：与代码仓库、CI/CD、Confluence、测试管理深度集成，Atlassian Marketplace 数千插件。
