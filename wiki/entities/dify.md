# Dify

> 开源低代码 AI 应用平台，强调可视化定义 agentic workflow、连接已有工具和数据源，快速交付 AI 应用。

## 关键信息

- **类型**: 开源低代码 AI 应用平台
- **开发商**: Dify.ai 社区
- **核心定位**: AI 应用快速交付平台
- **抽象层级**: 高（可视化优先）
- **开发方式**: 可视化优先，低代码为主
- **默认交付模式**: 自托管开源平台或平台化部署

## 核心特色

| 特性 | 描述 |
|------|------|
| 可视化编排 | 拖拽式 workflow 设计器 |
| 知识库管理 | 文档上传、切片、RAG 集成 |
| 工具集成 | 内置 API 工具，支持自定义工具 |
| 模型接入 | 模型中立，支持多种 provider |
| 应用发布 | 可发布为 API 或 Web 应用 |

## 在 Agent 平台生态中的位置

Dify 位于 **应用交付层**，与 Gemini Enterprise Agent Platform 同层。相比 LangGraph（编排层）和 Kubeflow（基础设施层），Dify 更偏业务可视化，上手速度最快。

## 对比维度

- **vs Gemini**: Dify 更偏开源自托管、低代码；Gemini 更偏托管代码优先、GCP 深集成
- **vs LangGraph**: Dify 更快交付应用；LangGraph 更深控制执行
