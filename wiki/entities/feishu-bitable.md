# 飞书多维表格 (Bitable / Base)

> 飞书提供的业务数据管理与低代码应用搭建工具，介于电子表格、数据库、项目管理工具和低代码平台之间，支持字段类型、多视图、表间关联、权限、自动化、仪表盘和开放 API。

## 关键信息

- **类型**: 低代码业务系统搭建平台 / 在线数据库
- **开发商**: 字节跳动（飞书）
- **核心定位**: 把部门级业务从散乱表格升级为结构化数据协作系统
- **数据模型**: App → 数据表 → 记录/字段 → 视图 → 仪表盘 → 自动化流程
- **典型场景**: 项目管理、CRM、工单管理、招聘跟踪、内容排期、资产盘点

## 核心特色

1. **多维字段系统**：20+ 字段类型（文本/数字/日期/人员/关联/公式/AI），字段类型越明确自动化越稳定。
2. **六种视图**：表格/看板/甘特/日历/表单/画册，同一份数据服务不同角色。
3. **表间关联**：通过关联字段连接不同数据表（如客户表→合同表→回款表）。
4. **自动化流程**：数据变更触发通知/HTTP 请求/AI 生成，低代码工作流。
5. **开放 API**：服务端 API 支持批量读写（单次 1000 条），可做轻量数据后台。
